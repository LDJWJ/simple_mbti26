# 간단한 MBTI 테스트

React + Vite + Tailwind CSS로 만든 모바일 친화적인 MBTI 성격 테스트 웹 앱입니다.
성경 속 닮은꼴 인물을 친근한 일러스트와 함께 확인할 수 있습니다.

**배포 URL**: https://ldjwj.github.io/simple_mbti26/

## 기능

- 간단 테스트 (12문항)
- 모바일 반응형 디자인
- 16가지 MBTI 유형 상세 설명 (11줄)
- 유형별 성경 속 닮은꼴 인물 + SVG 일러스트
- AI 성격 분석 (Gemini 2.0 Flash Lite API)
- 결과 복사 (요약 / 상세)
- URL 파라미터로 결과 직접 접근 (`?type=INTJ`)
- 세부 점수(E/I, S/N, T/F, J/P) 시각화

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

## 배포

### GitHub Pages 배포

```bash
npm run build
npx gh-pages -d dist
```
